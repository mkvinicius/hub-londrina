Preciso criar o Painel Administrativo do Hub Londrina — acessível em /admin.
É a visão 360 do dono da plataforma. Nenhuma rota pública é afetada.

## Stack do projeto
- Frontend: React + Vite + Wouter + Tailwind CSS
- Backend: Express.js (Node) + TypeScript
- Banco: PostgreSQL + Drizzle ORM
- Monorepo: pnpm workspaces

---

## 1. PROTEÇÃO DA ROTA

Criar middleware de autenticação simples no Express:
- Variável de ambiente ADMIN_PASSWORD no .env
- Rota POST /api/admin/login recebe { password }
  → se correta, retorna { token } (JWT assinado com JWT_SECRET do .env)
  → se errada, retorna 401
- Todas as rotas /api/admin/* exigem header Authorization: Bearer <token>
- No frontend, guardar o token no localStorage sob a chave "hub_admin_token"
- Se não autenticado, redirecionar para /admin/login

## 2. ROTAS DO BACKEND — /api/admin/*

### GET /api/admin/stats
Retorna:
{
  totalBusinesses: number,
  byPlan: { free: number, destaque: number, premium: number },
  totalClicks: number,          -- soma de businesses.clicks
  totalWhatsappClicks: number,  -- soma de businesses.whatsappClicks
  recentSignups: number,        -- cadastros nos últimos 30 dias
  byZone: { norte: n, sul: n, leste: n, oeste: n, centro: n }
}

### GET /api/admin/businesses
Query params: page, limit, search, zone, planType, isVisible
Retorna lista paginada com todos os campos de cada negócio

### PATCH /api/admin/businesses/:id
Body pode conter qualquer combinação de:
{ planType, isVisible, zone, verified }
Atualiza o negócio no banco

### DELETE /api/admin/businesses/:id
Remove o negócio

### GET /api/admin/categories
Lista todas as categorias com businessCount

### POST /api/admin/categories
Body: { name, slug, icon, color }
Cria nova categoria

### PATCH /api/admin/categories/:id
Body: qualquer campo da categoria
Atualiza categoria

### DELETE /api/admin/categories/:id
Remove categoria (só se não tiver negócios vinculados)

## 4. FRONTEND — páginas do painel

### /admin/login
- Campo de senha + botão entrar
- Chama POST /api/admin/login
- Salva token e redireciona para /admin

### /admin (dashboard principal)
Cards de métricas no topo:
- Total de negócios
- Por plano: Gratuito / Destaque / Premium
- Total de cliques
- Cliques no WhatsApp
- Cadastros nos últimos 30 dias

Gráfico de barras simples (pode ser CSS puro ou recharts):
- Negócios por zona

### /admin/negocios
Tabela com colunas: Nome | Zona | Plano | Cliques | WhatsApp | Visível | Ações
Filtros no topo: busca por nome, filtro por zona, filtro por plano
Ações por linha:
- Toggle visível/invisível (PATCH isVisible)
- Mudar plano (select inline: free/destaque/premium)
- Botão excluir (com confirmação)

### /admin/categorias
Lista de categorias com: Nome | Ícone | Cor | Negócios vinculados | Ações
Botão "Nova categoria" abre formulário inline
Ações: editar nome/cor/ícone, excluir (se vazia)
