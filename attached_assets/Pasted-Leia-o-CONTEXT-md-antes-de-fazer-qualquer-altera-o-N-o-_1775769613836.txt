### CORREÇÃO 4 — Zonas dos negócios seed

Todos os 20 negócios estão com zone='centro'.
Atualizar via SQL/Drizzle para refletir as regiões reais:

- Negócios com region='Zona Norte' → zone='norte'
- Negócios com region='Zona Sul' → zone='sul'
- Negócios com region='Zona Leste' → zone='leste'
- Negócios com region='Zona Oeste' → zone='oeste'
- Negócios com region='Centro' → zone='centro'
- Negócios com region='Gleba Palhano' → zone='sul'
- Negócios com region='Jardim Quebec' → zone='norte'

Executar UPDATE businesses SET zone = CASE
  WHEN region = 'Zona Norte' THEN 'norte'
  WHEN region = 'Zona Sul' THEN 'sul'
  WHEN region = 'Zona Leste' THEN 'leste'
  WHEN region = 'Zona Oeste' THEN 'oeste'
  WHEN region = 'Gleba Palhano' THEN 'sul'
  WHEN region = 'Jardim Quebec' THEN 'norte'
  ELSE 'centro'
END;
