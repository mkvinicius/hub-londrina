## 2. SITEMAP.XML DINÂMICO

Criar rota GET /sitemap.xml no servidor que gera
o sitemap dinamicamente com todas as páginas do site.

No server.mjs ou em uma rota Express dedicada:

```javascript
app.get('/sitemap.xml', async (req, res) => {
  try {
    // Buscar todos os negócios ativos
    const businesses = await fetch(
      `http://localhost:${API_PORT}/api/businesses?limit=1000&status=active`
    ).then(r => r.json());

    const baseUrl = 'https://www.hublondrina.com.br';
    const today = new Date().toISOString().split('T')[0];

    // Páginas estáticas
    const staticPages = [
      { url: '/', priority: '1.0', changefreq: 'daily' },
      { url: '/categorias', priority: '0.8', changefreq: 'weekly' },
      { url: '/busca', priority: '0.8', changefreq: 'daily' },
      { url: '/anuncie', priority: '0.7', changefreq: 'monthly' },
      { url: '/cadastro', priority: '0.6', changefreq: 'monthly' },
      { url: '/norte', priority: '0.8', changefreq: 'weekly' },
      { url: '/sul', priority: '0.8', changefreq: 'weekly' },
      { url: '/leste', priority: '0.8', changefreq: 'weekly' },
      { url: '/oeste', priority: '0.8', changefreq: 'weekly' },
      { url: '/centro', priority: '0.8', changefreq: 'weekly' },
    ];

    // Gerar XML
    const urls = [
      ...staticPages.map(p => `
  <url>
    <loc>${baseUrl}${p.url}</loc>
    <lastmod>${today}</lastmod>
    <changefreq>${p.changefreq}</changefreq>
    <priority>${p.priority}</priority>
  </url>`),
      ...(businesses.data || []).map(b => `
  <url>
    <loc>${baseUrl}/negocio/${b.id}</loc>
    <lastmod>${b.updatedAt ? b.updatedAt.split('T')[0] : today}</lastmod>
    <changefreq>weekly</changefreq>
    <priority>0.6</priority>
  </url>`)
    ].join('');

    const xml = `<?xml version="1.0" encoding="UTF-8"?>
<urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">
${urls}
</urlset>`;

    res.set('Content-Type', 'application/xml');
    res.send(xml);
  } catch (e) {
    console.error('Sitemap error:', e);
    res.status(500).send('Sitemap generation failed');
  }
});
```

## 4. UPLOAD DIRETO DE MÍDIA NOS PRODUTOS

Atualmente os produtos da vitrine pedem URL externa.
Adicionar opção de upload direto de arquivo.

### Backend
POST /api/lojista/upload/product-media
- Aceita multipart/form-data com campo "file"
- Salva em /public/uploads/products/
- Retorna { mediaUrl: string, mediaType: 'image' | 'video' }
- Limite de tamanho: 10MB para imagens, 50MB para vídeos
- Tipos aceitos: image/jpeg, image/png, image/webp, video/mp4

```typescript
router.post('/upload/product-media',
  requirePlan('premium'),
  upload.single('file'),
  async (req, res) => {
    if (!req.file) {
      return res.status(400).json({ error: 'Nenhum arquivo enviado' });
    }

    const isVideo = req.file.mimetype.startsWith('video/');
    const maxSize = isVideo ? 50 * 1024 * 1024 : 10 * 1024 * 1024;

    if (req.file.size > maxSize) {
      fs.unlinkSync(req.file.path);
      return res.status(400).json({
        error: `Arquivo muito grande. Máximo: ${isVideo ? '50MB' : '10MB'}`
      });
    }

    const mediaUrl = `/uploads/products/${req.file.filename}`;
    const mediaType = isVideo ? 'video' : 'image';

    return res.json({ mediaUrl, mediaType });
  }
);
```
